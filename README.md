## Installation

```bash
$ npm install
```

## Iniciando aplicação

Primeiramente, é necessário mudar o nome do .env.example para .env.

Se a aplicação for iniciada via docker, nenhuma outra alteração precisa ser feita no .env.

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Via docker
```bash
# Inicia os containers
$ npm run docker:up

# Para os containers
$ npm run docker:down
```
